## Features

- Treesitter-based markdown parsing
- ASCII art headers with figlet
- Syntax highlighted code blocks
- Clean, minimal interface

## Navigation

- Space / PageDown: Next slide
- Backspace / PageUp: Previous slide
- q / Q: Quit presentation
- gg: First slide
- G: Last slide
